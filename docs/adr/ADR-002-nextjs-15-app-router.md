# ADR-002: Next.js 15 com App Router

| Campo | Valor |
|-------|-------|
| Status | **accepted** |
| Data | 2025-06 |

**Decisão:** Usar Next.js 15 com App Router como framework frontend.

**Justificativa:** SEO nativo via SSR/ISR, API routes para BFF, Server Components.

**Alternativas descartadas:** SPA pura (Vite + React), Remix.
