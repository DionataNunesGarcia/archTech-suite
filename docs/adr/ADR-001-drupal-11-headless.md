# ADR-001: Drupal 11 como Backend Headless

| Campo | Valor |
|-------|-------|
| Status | **accepted** |
| Data | 2025-06 |

**Decisão:** Usar Drupal 11 como CMS headless com JSON:API e GraphQL.

**Contexto:** O sistema precisa de content modelling flexível (20+ content types), suporte nativo a multisite, i18n, e um ecossistema de módulos maduro.

**Alternativas descartadas:** Strapi, WordPress + ACF.
